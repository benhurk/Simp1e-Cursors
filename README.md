# Simp1e-Cursors

**I'm not the theme's author.**

This repo only exists to make it easier to install this cursor theme from a script. The original release is available [here](https://www.pling.com/p/1932768).

```bash
git clone https://github.com/benhurk/Simp1e-Cursors.git
tar -xf Simp1e-Cursors/Simp1e-All.tar.xz -C ~/.icons/
rm -rf Simp1e-Cursors/
```
